# Permissions Tester

An educational diagnostic tool for iOS developers to understand and test app permissions.

## Overview

Permissions Tester helps developers and users:
- Understand iOS permission statuses (Granted, Denied, Limited, Restricted, Not Determined)
- Test permission behavior on devices with MDM profiles or parental controls
- Debug permission-related issues in their own apps
- Learn about all available iOS permissions and their purposes

## Features

- **Comprehensive Permission Testing**: Test 25+ iOS permissions including:
  - Location (Precise, Coarse, Always)
  - Camera & Microphone
  - Photos (Full & Limited Access)
  - Contacts, Calendar, Reminders
  - Health & Fitness data
  - Bluetooth (Central & Peripheral)
  - NFC, HomeKit, Notifications
  - App Tracking Transparency
  - And more...

- **Real-time Status Display**: See current permission status for each permission
- **Educational Information**: Learn what each permission does via Apple's official documentation
- **Offline Operation**: Works entirely offline with no data collection or transmission
- **Category Filtering**: Organize permissions by category for easier browsing

## Privacy & Data

**This app does NOT:**
- Collect, store, or transmit any personal data
- Use analytics or crash reporting
- Access any actual data from granted permissions
- Include advertisements or third-party tracking

The app only checks and displays permission status locally on your device.

## Supported iOS Versions

- iOS 26.0 and later

## Installation

Download from the App Store: [Permissions Tester](https://apps.apple.com/app/permissions-tester/id[YOUR-APP-ID])

## How to Use

1. Launch the app to see all available permissions
2. Use category filters to browse permissions by type
3. Tap any permission to view:
   - Current authorization status
   - Detailed description of what the permission does
   - Link to Apple's official documentation
   - Option to request the permission
4. Grant or deny permissions as prompted by iOS
5. Permission statuses update in real-time

## Use Cases

- **MDM Testing**: Verify permission behavior on devices with Mobile Device Management profiles
- **Parental Controls Testing**: Test how parental restrictions affect permission availability
- **Permission Debugging**: Diagnose why specific permissions aren't working in your app
- **Educational**: Learn about iOS permissions and privacy controls
- **App Review Preparation**: Understand which permissions your app needs before submission

## Permissions Reference

This app requests permissions **only for testing purposes**. No data is collected or transmitted. For detailed information about each permission, launch the app and tap any permission to view Apple's official documentation.
